# PortVia

PortVia is a dynamic full-stack personal portfolio website showcasing my projects and skills as a Full Stack & AI Developer. It features an integrated AI-powered chatbot for interactive user queries, dynamic content rendering, theme switching, and seamless navigation, demonstrating expertise in building responsive and engaging web applications.

## Features

- **Personal Portfolio Display**: Showcases projects, skills, certifications, internships, and LeetCode achievements.
- **Theme Switching**: Customizable UI themes for an enhanced user experience.
- **Responsive Design**: Accessible and optimized for desktop and mobile devices.
- **Interactive Sections**: Includes about me, skills, featured projects, experience, certifications, recent activity, and contact information.

## Tech Stack

- **Frontend**: Built with React and Next.js for dynamic user interfaces, styled with CSS.
- **Backend**: Node.js handling API endpoints and data processing.
- **Build Tools**: Vercel for deployment and optimization.

## Installation

To set up PortVia locally:

1. Clone the repository:
   ```
   git clone https://github.com/Dharaneesh05/PortVia.git
   ```
2. Navigate to the project directory:
   ```
   cd PortVia
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Set up environment variables: Create a `.env` file based on `.env.example` (if available) and add necessary keys (e.g., MongoDB URI, API secrets).
5. Start the development server:
   ```
   npm run dev
   ```
   - The app will run on `http://localhost:3000` (or similar).

For production build:
```
npm run build
npm run start
```

## Usage

- **Explore Portfolio**: View detailed sections for projects, skills, experience, and certifications.
- **Interact with AI Chatbot**: Ask questions about my work or get assistance via the integrated AI assistant.
